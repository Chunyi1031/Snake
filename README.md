# 贪吃蛇

## 版本 V1.1
## 编程语言:Python

## 依赖文件:`setting.json`,此文件为配置文件，以下是使用方法:
```json
{
    "fps":"帧率",
    "bgcolor":["背景R","G","B"],
    "snakecolor":["蛇R","G","B"],
    "startxy":["起始x","y"],
    "full_marks":"满分"
}
```
### 实际修改时参数必须为整数且不加引号

## 使用方法
- 源代码可以使用pythonw运行，也可以运行安装程序，仅限Win64
- 安装程序会将文件安装在`C:\Users\用户名\Documents\贪吃蛇`，且创建桌面快捷方式
- 使用WSAD键控制方向，撞墙或撞到自己判断死亡
